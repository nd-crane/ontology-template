# Ontology Design Pattern Based FAIR Modular Ontology Construction Template

Template for Modular Ontology Construction using GitHub Actions for Continuous Integration/Publication for Ontologies.

## Tools Used

- [MODL Ontology Design Pattern Library](https://github.com/Data-Semantics-Laboratory/modular-ontology-design-library)
- [Apache Jena Riot](https://jena.apache.org/documentation/tools/)
- [Apache Jena SHACL](https://jena.apache.org/documentation/shacl/index.html)
- [Widoco](https://github.com/dgarijo/Widoco)
- [RDFX](https://github.com/surroundaustralia/rdfx)
- [Test Anything Protocol](https://testanything.org)
- [Sharness](https://github.com/chriscool/sharness)
- [GitHub Actions](https://docs.github.com/en/actions)
